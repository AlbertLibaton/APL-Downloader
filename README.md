Project Demo Overview: APL Downloader (v1.0.12)
APL Downloader is a unified, lightweight utility for macOS designed for seamless media downloading and high-fidelity audio playback. Built with a clean, dark-themed user interface, it automates link grabbing from the clipboard and handles both high-resolution video streams and high-quality audio files.

Detailed Walkthrough & Features
1. Intuitive Video Downloader & Clipboard Integration
Automated Link Detection: Clicking "Enter or Open URL" opens a dedicated clipboard analyzer. It instantly reads a copied media link and presents specific quality presets tailored to the source.

Smart Quality Selection: * Video: Supports multi-tier downloads, ranging from fast 708p/1080p AVC (Balanced) up to 4K AV1 (Best Quality) configurations.

Audio: Includes dedicated extraction options like high-quality Opus Audio (HQ).

Real-time Progress Monitoring: The main interface dynamically displays the media title, active download speeds (in MB/s), current percentage, file size tracking, and a visual progress bar.

Background Multiplexing: The application seamlessly handles fragmented downloads (combining separate video and audio streams into a single final container) directly in the background.

2. Multi-Platform Support
The demo showcases robust parsing across multiple popular video and streaming platforms:

YouTube: Full-length video scraping with fast download throughput.

YouTube Music: Smooth extraction of standalone high-quality audio tracks directly to the local drive.

Bilibili: Seamless URL parsing and download execution for international media sources.

3. Integrated Hi-Res Music Player
Switching modes transforms the app into a native audio player, optimized for music library management:

Audio Engineering Details: Supports bit-perfect tracking displays (e.g., 16-bit / 44.1 kHz stream confirmation).

Playback & Playlist Controls: Includes full playback manipulation (Play, Pause, Track Skipping), random sorting (Shuffle On/Off), and an interactive Playlist drop-down menu to quickly navigate through entire directories.

4. System Utility & Support
Update Engine: An integrated version checker ("Check for Updates") provides immediate, clean UI dialog prompts verifying software status.

Dedicated Help Ecosystem: The "Help & Support" action securely bridges the desktop environment to official project channels or developer documentation via the web browser.
